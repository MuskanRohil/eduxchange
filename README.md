Educxchange is a Peer-to-Peer Skill Exchange Platform. It is like a barter system for education where instead of money, people trade their knowledge.

It is basically a simple web application that helps people learn and teach skills by connecting them with each other. The main idea of this project is to make learning easier by allowing users to share their knowledge and learn from others. Users can create an account, log in securely, and manage their profiles. They can add the skills they know and also see skills from other users. The system also helps in finding matching users based on what a person wants to learn and what others can teach. In addition, users can send session requests to each other for learning, and they can accept or reject these requests. After completing a session, users can give ratings and feedback.

The backend of this project is built using Java and Spring Boot, and it uses MySQL to store data. The frontend is created using React.js making it easy to use for anyone. The application is tested using Postman, and GitHub is used to manage the project code.

Technologies used:

Java, Spring Boot
Spring Security (JWT Authentication)
MySQL
HTML, CSS, JavaScript
Postman
GitHub

Key features:

* Secure login and registration system
* Add and view skills
* Find matching users based on skills
* Send, accept, and reject session requests
* Give ratings and feedback
* Simple and easy-to-use interface

Overall, EduXchange is designed to make skill sharing simple, fast, and useful for everyone.
